# Веб-парсер цитат на Python (Selenium)

Скрипт для автоматического сбора данных с веб-ресурсов, использующих защиту от автоматических запросов. Обходит блокировки роботов с помощью имитации поведения реального пользователя в браузере Chrome и выгружает итоговые данные в структурированный файл Excel.

## 🚀 Основной функционал
* **Обход защиты:** Автоматическая маскировка под реального пользователя (отключение флагов автоматизации WebDriver).
* **Сбор данных:** Извлечение текста цитат и авторов со страницы в реальном времени.
* **Экспорт:** Формирование структурированного отчета в формате `.xlsx` (Excel).

## 🛠 Стек технологий
* **Python 3.x**
* **Selenium WebDriver** (автоматизация браузера)
* **OpenPyXL** (работа с Excel-таблицами)

## 📦 Как запустить проект

1. Клонируйте репозиторий или скачайте код.
2. Установите необходимые библиотеки одной командой:
   ```bash
   pip install selenium openpyxl
   ```
3. Запустите скрипт:
   ```bash
   python selenium_parser.py
   ```

## 📊 Результат работы
После успешного завершения работы скрипта в папке проекта создается файл `quotes_selenium.xlsx` со следующей структурой:
* **Колонка А:** Текст цитаты
* **Колонка Б:** Автор цитаты
